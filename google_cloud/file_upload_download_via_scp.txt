### Upload
    $ scp -v index.html asifulmamun@34.130.90.170:/var/www/html
        Where:
            scp - command
            -v - (if -r it will transfer folder) example:
                $ scp -r index.html asifulmamun@34.130.90.170:/var/www/html
            index.html - file name/directory name or path (can use /home/asifulmamun/desktop/myfile.txt)
            asifulmamun@34.130.90.170 - before colon ':', server destination
            /var/www/html - after colon ':', destination of directory, where file upload
